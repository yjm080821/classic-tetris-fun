# classic-tetris-fun

## 📌 프로젝트 목적
Lovable에서 생성/관리되는 프론트엔드 프로젝트로,  
로컬 개발(IDE) 또는 Lovable 웹에서 수정하고 배포까지 진행할 수 있도록 구성된 프로젝트입니다.

## 🧠 사용 기술
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
  > 레포 README에 명시된 기술 스택 기준

## ⚙ 구현 내용
- Lovable 프로젝트 페이지에서 프롬프트로 수정하면 변경 사항이 자동 커밋되는 흐름 지원
- 로컬 개발 환경에서 npm 설치 후 개발 서버 실행 가능
- Lovable에서 Share → Publish로 배포 가능

## 📁 프로젝트 구성(레포 기준)

* `public/`
* `src/`
* `index.html`
* `vite.config.ts`
* `tailwind.config.ts`
* `eslint.config.js`
* `package.json` 등

## ✍ 느낀 점

* Vite + React + TypeScript + Tailwind(shadcn-ui 포함) 조합으로 프론트엔드 프로젝트를 빠르게 구성할 수 있음을 확인했습니다.
* Lovable을 통해 코드 수정 → 자동 커밋 → 배포까지 이어지는 워크플로우를 사용할 수 있다는 점이 인상적이었습니다.
